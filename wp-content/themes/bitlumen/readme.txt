== Description ==
BitLumen is a compact, clean, adjustable fixed-width, traditional style theme. It has many powerful theme options that empower you to uniquely brand your site, including custom headers, text, colors, and backgrounds. It also utilizes microdata/microformats and integrated meta descriptions for easy, out-of-the-box SEO performance.
