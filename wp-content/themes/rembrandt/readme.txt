Requires at least: WP 3.4

== Description ==
Minimalistic two-columns, fixed-width,  seo optimized  theme for WordPress. Compatible with WordPress 3.1 features, valid XHTML & CSS, mobile version, custom widgets.
Rembrandt theme includes support for the following languages:
* English
* Polish
