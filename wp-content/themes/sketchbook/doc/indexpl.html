A) Instalacja - top

Gdy masz juz gotową instalację i chcesz dodać motyw, najpierw musisz go załadować a następnie aktywować. Można to zrobić na dwa sposoby:

  • FTP: skopiuj programem ftp (np FileZilla) nieskompresowany katalog z motywem do folderu /wp-content/themes/ na twoim serwerze.
  • Panel WordPress: Przejdź do Appearance > Add New Themes > Upload. Przejdź do przzeglądaj i wybierz skompresowany folder z motywem. wybierz "Install Now" i motyw zostanie zainstalowany.

Po przesłaniu motywu trzeba go aktywować, przejdź do Appearance > Themes i aktywuj szablon.

D) Opis kodu PHP - top

Oprócz standardowych plików szablonu, w motywie znajdują się również pliki odpowiadające za konfigurację oraz niestandardowe funkcje rozszerzające możliwości szablonu. Poniżej krótkie omówienie:

  • folder /admin - pliki odpowiadające za opcje motywu - Options Framework - 0.8 przez Devin Price,
  • options.php - plik odpowiadający za konfiguracje opcji motywu,
  • folder /doc - dukumentacja,
  • folder /inc - pliki z niestandardowymi funkcjami,
    • post-options-api.1.0.1.php - opcje wpisu - Post Options API 1.0.1 przez Konstantin Kovshenin,
    • sketchbook-post-options.php - funkcje odpowiadające ze wdrożenie ustawien z "Post Options" ,
    • shortcode.php - shortcode i klasy,
    • sketchbook-functions.php - funkcje odpowiadające ze wdrożenie ustawień z "Options Framework",
    • widgets.php - widgety,
  • folder /js - pliki javascript (html 5, jquery fancybox, jquery flexslider, adapt).

E) Pliki CSS - top

Motyw został zbudowany na szablonie css 960 Grid System. Całkowita szerokość wynosi 960px. Aby uzyskać więcej informacji o sposobie działania siatki przejdź do strony 960 Grid System website.

  • folder /css - wersja do druku, grid 960px, wersja mobilna, style edytora, wtyczki (lightbox, slideshow),
  • rtl.css - wersja z wyrównaniem do prawej (j. arabskie),
  • style.css - główny plik ze stylami.

F) JavaScript - top

Motyw zawiera cztery pliki javascript.

  1. html5.js,
  2. adapt.min.js,
  3. jquery.fancybox,
  4. jquery.flexslider-min.js.
